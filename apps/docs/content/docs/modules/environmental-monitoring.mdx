# Environmental Monitoring

Monitor cleanroom environments per USP &lt;797&gt; requirements — viable and non-viable sampling, trend analysis, and excursion management.

## Features

- Room map visualization with ISO classification
- Sampling schedule calendar
- Viable count data entry and trending
- Alert/action limit tracking
- Automatic excursion detection
- Batch impact assessment

## Excursion Flow

```
Sample Collected <Icon name="ArrowRight" size={12} /> Results Entered <Icon name="ArrowRight" size={12} /> Limits Checked <Icon name="ArrowRight" size={12} /> Excursion Created (if exceeded) <Icon name="ArrowRight" size={12} /> Deviation Linked <Icon name="ArrowRight" size={12} /> Corrective Action
```
